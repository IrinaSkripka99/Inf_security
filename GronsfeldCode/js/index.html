Gronsfeld

Шифр Гронсфельда был создан Хосе де Бронкхорстом (графом Гронсфельдом) около 1744 года. Как бельгийский дипломат, безопасность сообщений при обмене была для него очень важной.  Шифр Гронсфельда представляет собой разновидность шифра Vigenere, в котором вместо ключевого слова используется номер ключа, например 14965.  Для шифрования с использованием этого шифра буква текста берется одна за другой и сдвигается на соответствующий номер в ключе. Ключ повторяется, чтобы соответствовать длине сообщения. Например:

Текст : NAT

Ключ : 123

Зашифрованный текст : OCW

Этот код не предлагает никакой защиты. Однако сломать сложнее Шифра Цезаря.
